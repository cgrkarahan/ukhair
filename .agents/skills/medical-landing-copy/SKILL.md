# Medical Landing Copy

Use this skill when a page needs to convert without sounding pushy, cheap, or vague.

## Initial Check

If `.agents/product-marketing-context.md` exists, read it first and use it as the baseline positioning and tone context.

## Page Structure

Default order:

1. Clear value proposition
2. Trust / standards
3. Suitability and planning
4. Method / pathway clarity
5. Pricing logic
6. Recovery / objections
7. CTA

## Tone

- calm
- premium
- adult
- medically literate
- direct

Avoid:
- chatty sales language
- artificial luxury clichés
- hype-heavy bargain framing
- startup SaaS CTA patterns

## Conversion Rules

- One primary CTA per page.
- Secondary CTA should support research, not distract.
- Put trust near the hero.
- Put objections near the CTA.
- Explain what happens next.

## Hair-Transplant Specific Objections

Always consider:
- is this right for me?
- who is performing it?
- how much will it cost?
- what does recovery look like?
- why stay in London instead of Turkey?
- what if I want subtlety rather than density?

## Preferred Output

When rewriting a page, default to:
- hero copy
- trust block
- objection-handling section
- CTA copy
- short rationale for why the structure should convert
