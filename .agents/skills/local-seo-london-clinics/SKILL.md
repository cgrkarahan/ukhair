# Local SEO London Clinics

Use this skill for authentic local visibility rather than location-page spam.

## Initial Check

If `.agents/product-marketing-context.md` exists, read it first and use it as the baseline business context.

## Strategy Order

1. Strengthen one primary London commercial page.
2. Add only authentic supporting location pages.
3. Reinforce local trust signals across the site.
4. Keep internal links and entity signals consistent.

## Local Page Rules

- Prefer one strong `Hair Transplant London` page over many weak borough pages.
- Only create extra location pages when there is real service logic, unique copy, and clear user value.
- Use `central London` / `Harley Street` references carefully and truthfully.
- Avoid cloning near-identical city pages.

## On-Page Priorities

- clear London intent in title/H1/body
- local trust language
- access / convenience / aftercare relevance
- internal links to cost, standards, and treatment pages

## Non-Page Signals

Consider:
- business profile consistency
- contact and location details
- local citations / mentions
- maps and service-area clarity

## Preferred Output

When producing recommendations, default to:
- primary London page strategy
- supporting local-page plan only where justified
- on-page local signals
- off-page / entity consistency checks
