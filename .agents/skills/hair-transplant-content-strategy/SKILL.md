# Hair Transplant Content Strategy

Use this skill to turn keyword and intent data into a ranking plan that also supports conversion.

## Initial Check

If `.agents/product-marketing-context.md` exists, read it first and use it as the baseline business and audience context.

## Core Clusters

Always map research into these buckets first:

1. Commercial
   - London
   - cost
   - male / female
   - treatment pages
2. Decision
   - UK vs Turkey
   - method comparisons
   - suitability
3. Recovery
   - timeline
   - side effects
   - donor area
   - aftercare
4. Trust
   - standards
   - clinic selection
   - patient guidance
5. Local
   - London-first
   - only add more locations if authentic

## Planning Rules

- One primary URL per topic.
- Avoid multiple weak pages targeting the same head term.
- Use high-intent informational pages to feed commercial pages.
- Use recovery content as a moat, not a blog afterthought.
- Keep female demand visible even if male demand is larger.
- Keep Turkey balanced and future-compatible.

## Default Deliverables

- sitemap / route map
- page priority list
- cluster map
- internal-link plan
- content brief per page
- cannibalization warnings where relevant

## Internal Linking Pattern

Commercial pages should point to:
- cost
- standards
- recovery
- service pages
- method pages

Informational pages should point back to:
- London commercial page
- relevant treatment page
- assessment CTA

## Preferred Output

When possible, deliver:
- primary keyword / intent
- recommended URL
- page role in the funnel
- internal links in and out
- proof or trust requirements needed to compete
