# Premium Healthcare Branding

Use this skill to translate a strategic position into a visual and verbal brand direction that feels premium, credible, and commercially useful.

## Initial Check

If `.agents/product-marketing-context.md` exists, read it first and use it as the baseline business, audience, and tone context.

## Brand Questions To Answer

1. What should the brand feel like in the first five seconds?
2. What trust signals must be visible early?
3. What visual clichés would make the brand feel cheaper, louder, or less medically credible?
4. How should the brand distinguish itself from generic brokers, cosmetic chains, and flashy “Turkey deal” sites?

## Default Direction For This Project

- premium but not theatrical
- clinical but not cold
- editorial but not magazine-fashion
- masculine-leaning demand aware but not exclusionary
- London-serious rather than bargain-tourism-led

## Core Brand Ingredients

- strong typography hierarchy
- restrained but intentional color system
- visible trust signals and standards
- precise copy rather than slogan-heavy fluff
- real process and recovery cues, not only vanity imagery

## Avoid By Default

- overused luxury black/gold tropes
- stock-photo-heavy spa aesthetics
- startup SaaS visual language
- overly aggressive “best clinic” branding
- macho or laddish hair-loss marketing

## Preferred Output

When asked to help, default to:
- brand direction summary
- visual guardrails
- tone guardrails
- homepage brand expression notes
- examples of what to emphasize and what to avoid
