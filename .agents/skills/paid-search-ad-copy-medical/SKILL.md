# Paid Search Ad Copy Medical

Use this skill when paid-search ads need to be persuasive without slipping into risky or low-trust medical marketing language.

## Initial Check

If `.agents/product-marketing-context.md` exists, read it first and use it as the baseline positioning and trust-language context.

## Copy Priorities

1. Intent match
2. Trust and standards
3. Clarity on what happens next
4. Strong but defensible CTA

## Good Angles For This Project

- premium clinic in central London
- GMC-registered doctors
- CQC-registered provider in England
- consultation / assessment pathway
- transparent planning and aftercare

## Avoid By Default

- fixed-price dependence
- anti-Turkey framing
- unsupported superlatives
- sensational before/after claims
- vague “best clinic” language
- cheap, bargain-heavy framing

## Asset Types To Think Through

- RSA headlines
- RSA descriptions
- sitelinks
- callouts
- structured snippets where relevant

## Preferred Output

When asked to help, default to:
- headline bank
- description bank
- asset extensions
- angle-by-angle rationale
- compliance notes for risky lines
