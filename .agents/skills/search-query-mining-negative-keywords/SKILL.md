# Search Query Mining and Negative Keywords

Use this skill to keep search budgets focused on patient-relevant intent.

## Initial Check

If `.agents/product-marketing-context.md` exists, read it first and use it as the baseline commercial and measurement context.

## Review Order

1. Identify queries with no patient-intent value.
2. Separate research intent from consult-ready intent.
3. Find themes worth negating at account, campaign, or ad-group level.
4. Promote genuinely strong queries into clearer keyword buckets where useful.

## Common Waste Themes To Check

- jobs and careers
- training and courses
- DIY / home methods
- unrelated products
- celebrity or gossip searches
- irrelevant countries or locations
- very low-intent “what is” traffic if the campaign is meant to convert now

## Output Preference

When asked to help, default to:
- negative-keyword recommendations
- wasted-spend themes
- query re-bucketing suggestions
- expansion opportunities
- weekly review cadence
