# Creative Direction QA

Use this skill to evaluate whether the executed design matches the intended brand, trust level, and conversion strategy.

## Initial Check

If `.agents/product-marketing-context.md` exists, read it first and use it as the baseline brand and audience context.

## Review Questions

1. Does the page feel premium, clinical, and trustworthy within seconds?
2. Does the visual tone match the copy tone?
3. Are trust cues prominent enough for a medically sensitive decision?
4. Does the design feel distinctive, or like a recycled clinic template?
5. Do the CTA patterns support the intended conversion flow?

## Common Failure Modes

- design looks luxurious but not medically credible
- design looks safe but generic
- trust cues are too weak or too late
- typography and spacing feel inconsistent
- mobile experience undermines the premium impression

## Preferred Output

When asked to help, default to:
- top issues by priority
- what is on-brand
- what is off-brand
- concrete design adjustments
- residual risks after revision
