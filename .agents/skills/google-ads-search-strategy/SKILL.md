# Google Ads Search Strategy

Use this skill to build a paid-search structure that matches real patient intent and lead quality goals.

## Initial Check

If `.agents/product-marketing-context.md` exists, read it first and use it as the baseline positioning, audience, and measurement context.

## Strategy Order

1. Separate high-intent commercial themes from informational themes.
2. Prioritize the highest-quality search terms before expanding volume.
3. Match campaign structure to landing pages that can genuinely convert.
4. Protect budget with negatives and query discipline.

## Default Campaign Buckets

Start by separating:

- hair transplant london
- hair transplant cost london
- female hair transplant london
- FUE / DHI / method-led intent
- brand campaigns
- competitor campaigns only if explicitly desired

## Structure Rules

- Keep campaign themes narrow enough that ad copy and landing pages can stay specific.
- Use separate landing pages when the intent is meaningfully different.
- Do not send all traffic to the homepage by default.
- Split high-risk or experimental themes away from core budget.

## Bidding and Expansion

- Start with lead-quality-friendly intent, then test broader reach carefully.
- Prefer expansion only after conversion tracking and negative-keyword controls are sound.
- Judge success on qualified consultations, not form count alone.

## Preferred Output

When asked to help, default to:
- campaign map
- ad-group or theme structure
- keyword buckets by intent
- landing-page mapping
- negative-keyword risk notes
- budget or bidding guidance if requested
