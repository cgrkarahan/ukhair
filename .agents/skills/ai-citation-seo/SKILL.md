# AI Citation SEO

Use this skill to make pages easier for answer engines to quote, summarize, and cite.

## Initial Check

If `.agents/product-marketing-context.md` exists, read it first and use it as the baseline positioning and trust context.

## What Helps Citation

- direct answers near the top
- question-like headings
- short summary paragraphs
- tables and comparison blocks
- FAQ sections
- explicit limitations and caveats
- consistent entity signals

## What Usually Fails

- vague brand copy
- unsupported superlatives
- pages that never answer the question directly
- important facts buried in decorative sections

## Default Checklist

1. Can the first 150 words answer the query?
2. Are headings phrased around user questions?
3. Are definitions, comparisons, and timelines easy to extract?
4. Is the page internally linked to trust and supporting evidence?
5. Are major crawlers allowed if desired?

## Preferred Output

When improving a page, produce:
- answer block
- supporting detail block
- FAQ block
- citation-friendly table or bullets
- crawler or discoverability notes if relevant
